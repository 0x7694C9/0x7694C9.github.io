Commands

This page lists key commands. For full details, consult the support server or use /help in Discord.

Moderation

  • ban • Ban a user
  • kick • Kick a user
  • purge • Bulk delete messages
  • unban • Unban a user

Utility

  • ping • Check latency
  • serverinfo • Server information
  • userinfo • User profile info
  • weather • Weather lookup
  • time • Show current time in a specified area

Fun

  • 8ball • Ask the magic 8-ball

Music | Very WIP

  • play • Play a song
  • pause • Pause the current song
  • resume • Resume the paused song
  • skip • Skip the current song
  • stop-song • Stop the current song
  • nowplaying • View the current playing song

Miscellaneous

  • embed • Creates an embedded message
  • botinfo • Bot information
  • afk • Set your status to AFK
