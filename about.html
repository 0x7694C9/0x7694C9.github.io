About Aurora

Aurora is a Discord bot focused on moderation and utility features with an emphasis on ease-of-use and reliability.

Features

  • Moderation commands and logging
  • Slash commands and traditional prefixed commands
  • Extensible architecture with config-driven options

Contribute

Aurora is closed-source. If you'd like to contribute or discuss partnerships, please contact the project owner via the support server or contact channel.
