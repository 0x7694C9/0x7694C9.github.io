Privacy Policy

Last updated: February 2026

Data We Receive

Aurora processes the data necessary to provide bot features: usernames, IDs, messages (when commands interact), and server configuration. We do not collect more than needed.

How Data Is Used

Data is used to run commands, moderation, logging (if enabled), and to provide configured features. We do not sell personal data.

Retention

Retention depends on feature settings (e.g., moderation logs). Server owners control configuration and may delete logs via bot commands where available.

Third Parties

The bot may communicate with third-party APIs (e.g., weather). Data sent to third parties is limited to what's required for that service.

Contact

Questions or requests about privacy should be sent to the project owner via GitHub or support server.
